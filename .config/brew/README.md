

### パッケージdump

```bash
brew bundle dump --file=MyBrewfile
```

`-f`オプションでファイルを上書きする

### パッケージインストール

```bash
brew bundle --file=MyBrewfile
```


## 運用

### パッケージdump

```bash
brew bundle dump --file="~/.config/brew/Brewfile"
```

### パッケージインストール

```bash
brew bundle --file="~/.config/brew/Brewfile"
```
