# Tauri IPC パターン

## 使用タイミング

Tauri コマンドやフロントエンド側の invoke 呼び出しを追加/変更するときに使用する。

## コア規約

- フロントエンドのパラメータ名は `camelCase` を使う。
- Rust のコマンドハンドラは `snake_case` を使う。
- 複雑化してきたら IPC DTO とドメインモデルを分離する。

## フロントエンド側

- `@tauri-apps/api/core` から `invoke` をインポートする。
- コマンドエラーを一貫した方針で扱い、型付き結果を返す。

## Rust 側

- コマンド境界で ID を検証/変換する。
- コマンドハンドラは薄く保ち、業務ロジックはサービス/ファサードへ委譲する。
- フロントエンド実装を単純化するため、成功/失敗の戻り値形状を予測可能にする。
