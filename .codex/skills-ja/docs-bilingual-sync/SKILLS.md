# バイリンガル文書同期

## 使用タイミング

2つの言語版を持つドキュメントを新規作成または更新するときに使用する。

## ワークフロー

1. 原文ドキュメントと対応する他言語版ドキュメントを特定する。
2. 構造と内容が等価になるように両方へ更新を適用する。
3. 見出し、例、コマンドスニペットの整合性を確認する。
4. 意図的な差異がある場合は、その理由を明示的に記録する。

## 品質チェック

- 用語の使い方を両言語で一貫させる。
- 対応する言語パスがある場合、リンク先も対応版に合わせる。
- ユーザーが明示的に依頼しない限り、片側だけ更新しない。
